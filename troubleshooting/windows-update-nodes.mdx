## Clash-Windows 更新节点

### Clash for Windows 更新节点方法

1. 选择【General】
2. 将 【System Proxy】开关关闭状态

![Clash for Windows 更新步骤1](/images/windows-setup-1.webp)

1. 选择【Profiles】
2. 点击 【user.crowvpn.com】（subscribe）项的 刷新图标
3. 更新完成，重新打开第一步关闭的 System Proxy 开关，就可以上网了

![Clash for Windows 更新步骤2](/images/windows-setup-2.webp)

PS：刷新图标转一会，就停下，说明已经更新成功。

<Warning>
  **出现 Error 弹窗，说明没有更新成功，确定以下：**
  
  1. System Proxy 开关是否关闭状态
  2. 查看网站的套餐流量和有效期是否正常范围
</Warning>
