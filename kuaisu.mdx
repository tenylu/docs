### 为什么要重新配置？

平台节点会被GFW大数据封堵，直接把配置请求服务器ip 墙了，所以无法请求更新，需要更换链接解决。

## <Tooltip tip="macos">MacOS解决方法</Tooltip>

打开 ClashX

1.关闭设置为系统代理 -\> 2.选择 配置-托管配置-管理

![1280X1280 (4).PNG](/images/1280X1280\(4\).PNG)

3.编辑链接 -\> 4. 修改链接地址为 { [<u>https://user.crowmesh.com</u>](https://user.crowmesh.com)<u> </u>} /api/v1.... .

<Warning>
  原来 cdn.niudache.com 地址 改为 **<u>user.crowmesh.com</u>**
</Warning>

![1280X1280 (5).PNG](/images/1280X1280\(5\).PNG)

确定后，重新打开 设置为系统代理 就可以用了

---

## Windows 解决方法

打开 Clash for windows

 1.选择General -\> 关闭 Proxy开关

![1280X1280 (6).PNG](/images/1280X1280\(6\).PNG)

2.选择ProFiles -\> 选择CrowVPN配置-\> 右键列表选择设置

3.. 修改URL 链接地址为 { <u>https://user.crowmesh.com</u> } /api/v1.... .

<Warning>
  原来 cdn.niudache.com 地址 改为 **<u>user.crowmesh.com</u>**
</Warning>

![07a9d475-31f5-41ba-b0bd-3082f28090ad.png](/images/07a9d475-31f5-41ba-b0bd-3082f28090ad.png)

OK 后，重新打开 打开Proxy开关 就可以用了

---

## iPhone/iPad 解决方法

打开 Shadowrocket (小火箭)

 1.关闭开关 -\> 选择 CrowVPN的 叹号图标

2. 修改URL链接地址为 { <u>https://user.crowmesh.com</u> } /api/v1.... .

<Warning>
  原来 cdn.niudache.com 地址 改为 **<u>user.crowmesh.com</u>**
</Warning>

![227f243c-2b71-4fc6-b1f3-ddf0a4b604ca.png](/images/227f243c-2b71-4fc6-b1f3-ddf0a4b604ca.png)

保存后，重新打开 开关 就可以用了
