### 安装说明

#### DEB 包安装方式（amd64、arm64）

适用于 Debian、Ubuntu 等基于 Debian 的发行版：

1. 下载 DEB 安装包

2. 使用 dpkg 安装：
   ```bash
   sudo dpkg -i crowvpn-linux-2.1.0-*.deb
   ```

3. 如果遇到依赖问题，运行：
   ```bash
   sudo apt-get install -f
   ```

#### RPM 包安装方式（x86_64、aarch64）

适用于 RHEL、CentOS、Fedora 等基于 RPM 的发行版：

1. 下载 RPM 安装包

2. 使用 rpm 或 dnf/yum 安装：
   ```bash
   # 使用 rpm 安装
   sudo rpm -i crowvpn-linux-2.1.0-*.rpm
   
   # 或使用 dnf/yum 安装（推荐）
   sudo dnf install crowvpn-linux-2.1.0-*.rpm
   # 或
   sudo yum install crowvpn-linux-2.1.0-*.rpm
   ```

#### 系统要求

- Linux 内核 3.10 或更高版本
- glibc 2.17 或更高版本
- **DEB 包**：Ubuntu 18.04+, Debian 9+ 等
- **RPM 包**：RHEL 7+, CentOS 7+, Fedora 28+ 等

<Info>
  请根据您的系统架构和发行版选择对应的安装包格式。DEB 包适用于 Debian/Ubuntu 系统，RPM 包适用于 RHEL/CentOS/Fedora 系统。
</Info>
