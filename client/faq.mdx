## 登录与配置问题

### 免配置登录使用

<AccordionGroup>
  <Accordion title="无法登录客户端">
    **可能原因：**
    - 账号或密码错误
    - 网络连接问题
    - 账号被禁用或过期
    - 服务器连接失败
    
    **解决方法：**
    1. 检查账号和密码是否正确
    2. 确认网络连接正常
    3. 前往用户面板确认账号状态
    4. 尝试重新登录
    5. 检查是否有防火墙阻止连接
  </Accordion>
  
  <Accordion title="登录后无法自动配置">
    **可能原因：**
    - 账号未激活或套餐已过期
    - 网络无法访问配置服务器
    - 客户端版本过旧
    
    **解决方法：**
    1. 确认账号状态和套餐是否有效
    2. 检查网络连接是否正常
    3. 更新客户端到最新版本
    4. 尝试手动刷新配置
    5. 联系技术支持
  </Accordion>
  
  <Accordion title="登录信息丢失">
    **可能原因：**
    - 客户端缓存被清除
    - 客户端被卸载重装
    - 系统设置重置
    
    **解决方法：**
    1. 重新输入账号密码登录
    2. 检查是否开启了"记住密码"选项
    3. 确认客户端已保存登录信息
  </Accordion>
</AccordionGroup>

## 连接相关问题

### 一键连接

<AccordionGroup>
  <Accordion title="点击一键连接无反应">
    **可能原因：**
    - 客户端未完成初始化
    - 配置未加载完成
    - 网络连接问题
    
    **解决方法：**
    1. 等待客户端完成初始化
    2. 检查配置是否已加载
    3. 确认网络连接正常
    4. 重启客户端后重试
  </Accordion>
  
  <Accordion title="连接失败">
    **可能原因：**
    - 节点服务器不可用
    - 网络环境限制
    - 防火墙阻止
    - 账号套餐已过期
    
    **解决方法：**
    1. 尝试切换其他节点
    2. 检查防火墙设置
    3. 确认系统时间是否正确
    4. 检查账号和套餐状态
    5. 尝试更换网络环境
  </Accordion>
  
  <Accordion title="连接成功但无法上网">
    **可能原因：**
    - DNS 解析问题
    - 代理模式设置错误
    - 路由规则问题
    
    **解决方法：**
    1. 检查代理模式设置（建议使用智能模式）
    2. 尝试刷新 DNS 缓存
    3. 切换到全局模式测试
    4. 重启客户端
  </Accordion>
</AccordionGroup>

### 智能模式切换

<AccordionGroup>
  <Accordion title="智能模式和全局模式的区别">
    **智能模式：**
    - 根据规则自动判断是否需要代理
    - 国内网站直连，国外网站走代理
    - 节省流量，速度更快
    
    **全局模式：**
    - 所有流量都通过代理
    - 适合访问所有网站都需要代理的场景
    
    **建议：** 日常使用推荐智能模式
  </Accordion>
  
  <Accordion title="智能模式无法正常工作">
    **可能原因：**
    - 规则未正确加载
    - DNS 设置问题
    - 客户端缓存问题
    
    **解决方法：**
    1. 检查规则是否已加载
    2. 刷新节点配置
    3. 清理客户端缓存
    4. 重启客户端
  </Accordion>
  
  <Accordion title="如何切换代理模式">
    **操作方法：**
    1. 打开客户端主界面
    2. 找到"代理模式"或"连接模式"选项
    3. 选择"智能模式"或"全局模式"
    4. 切换后会自动重新连接
  </Accordion>
</AccordionGroup>

## 节点相关问题

### 自动节点

<AccordionGroup>
  <Accordion title="自动节点不工作">
    **可能原因：**
    - 自动节点功能未开启
    - 节点列表为空
    - 网络无法访问节点服务器
    
    **解决方法：**
    1. 检查是否开启了自动节点功能
    2. 确认节点列表已加载
    3. 尝试手动刷新节点列表
    4. 检查网络连接
  </Accordion>
  
  <Accordion title="自动选择的节点速度慢">
    **可能原因：**
    - 当前节点负载过高
    - 网络环境不佳
    - 选择了距离较远的节点
    
    **解决方法：**
    1. 等待自动切换到其他节点
    2. 手动选择距离较近的节点
    3. 检查本地网络状况
    4. 尝试更换网络环境
  </Accordion>
  
  <Accordion title="如何手动选择节点">
    **操作方法：**
    1. 打开客户端主界面
    2. 进入"节点"或"服务器"页面
    3. 从节点列表中选择合适的节点
    4. 点击连接即可
  </Accordion>
  
  <Accordion title="节点列表为空或无法加载">
    **可能原因：**
    - 配置未正确加载
    - 订阅链接失效
    - 网络无法访问订阅服务器
    - 账号套餐已过期
    
    **解决方法：**
    1. 重新导入配置
    2. 检查订阅链接是否有效
    3. 确认账号和套餐状态
    4. 检查网络连接
    5. 尝试手动刷新节点列表
  </Accordion>
</AccordionGroup>

### 套餐信息查看

<AccordionGroup>

    2. 前往用户面板查看准确数据

## 共享代理问题

### 共享代理

<AccordionGroup>
  <Accordion title="什么是共享代理">
    **说明：**
    - 共享代理允许其他设备通过当前设备使用代理服务
    - 适合局域网内多设备共享使用
    - 需要客户端开启共享代理功能
  </Accordion>
  
  <Accordion title="如何开启共享代理">
    **开启方法：**
    1. 打开客户端
    2. 进入"设置" → "代理共享"或"共享代理"
    3. 开启共享代理功能
    4. 记录显示的代理地址和端口
    5. 在其他设备上配置该代理地址
  </Accordion>
  
  <Accordion title="共享代理无法使用">
    **可能原因：**
    - 防火墙阻止连接
    - 代理地址配置错误
    - 客户端未开启共享功能
    
    **解决方法：**
    1. 检查防火墙设置，允许客户端通过
    2. 确认代理地址和端口正确
    3. 确认共享代理功能已开启
    4. 检查设备是否在同一网络
  </Accordion>
</AccordionGroup>

## 安装与启动问题

### 安装问题

<AccordionGroup>
  <Accordion title="无法安装客户端">
    **Windows 系统：**
    - 检查系统版本（需要 Windows 10 或更高）
    - 临时关闭杀毒软件
    - 以管理员身份运行安装程序
    - 检查磁盘空间是否充足
    
    **macOS 系统：**
    - 前往"系统偏好设置" → "安全性与隐私" → 点击"仍要打开"
    - 如果仍无法打开，在终端运行：`xattr -cr /Applications/CrowVPN.app`
  </Accordion>
  
  <Accordion title="安装后无法启动">
    **解决方法：**
    1. 重新下载并安装客户端
    2. 检查系统版本是否满足要求
    3. 以管理员权限运行（Windows）
    4. 检查系统日志查看具体错误信息
  </Accordion>
</AccordionGroup>
